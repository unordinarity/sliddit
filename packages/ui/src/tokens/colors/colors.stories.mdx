# Colors

UI kit предоставляет два уровня цветов: богатую открытую палитру и готовые к использованию цветовые схемы. Что в них содержится и как это работает - описано ниже

## Theory

Существует несколько подходов по именованию цветов:

- Прямое название - в честь оттенка (Black, White, Purple, Gold, etc)
- Название по назначению (Comment, Stroke, etc)
- Название для состояния (Warning, Danger, etc)
- Градация по частоте (Primary, Secondary, etc)
- Цвета в зависимости от палитры (Accent, Brand, etc)

Эти подходы категорически запрещено перемешивать в коде, потому что некоторые из подходов формируют наборы цветов, пригодные лишь для дальнейших преобразований и не готовые для использования в коде.
